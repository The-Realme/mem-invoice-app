# MEM Office Suite

A professional Invoice & Quotation Management System developed for small businesses.

The application automates quotation and tax invoice generation with professional PDF templates, PostgreSQL storage, and dynamic customer management.

---

## Features

- Professional Quotation Generation
- Professional Tax Invoice Generation
- Automatic PDF Generation (Puppeteer)
- PostgreSQL Database
- Customer Management
- GST Calculations
- Bill To / Ship To
- Amount in Words
- Digital Signature Support
- Watermark & Letterhead
- Responsive Web Interface

## Installation

cd invoice-system

npm install

Create a PostgreSQL database.

Execute

database/schema.sql

Configure your database credentials.

Run

node server.js

Open

http://localhost:3000
